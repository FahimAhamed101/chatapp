✨ Key Features

🔐 JWT Authentication

Secure JWT-based authentication with role-based access control and token refresh mechanism.

💬 Real-time Messaging

Instant message delivery with WebSocket support and real-time read receipts.

👥 Conversation Management

1-to-1 conversations with search, filtering, and message history.

📱 Responsive Design

Mobile-friendly interface with Tailwind CSS and modern React patterns.

🗄️ MongoDB Integration

Scalable MongoDB database with optimized queries and proper indexing.

🚀 Production Ready

Error handling, validation, logging, and deployment-ready architecture.

📌 Important Notes

📁 File Locations: All HTML documentation files are in the root project directory: c:\Users\Fahim\Desktop\chat-system\

🌐 Viewing Documentation: Open any HTML file in your web browser to view beautifully formatted documentation with embedded CSS styling.

📚 User vs Sender: The system now uses "User" and "Sender" terminology throughout (previously "Patient" and "Doctor"). All documentation reflects this naming convention.

🚀 Getting to Localhost: When setting up, backend runs on http://localhost:5000, frontend on http://localhost:3000

💾 MongoDB Atlas: Ensure your MongoDB Atlas connection string is configured in .env before running backend.
